CSS Selectors

CSS Selectors let you select individual HTML elements in an HTML document. This is super useful.

Classes are very, very important, since they allow us to select arbitrary boxes in web pages.
Classes can be named nearly anything, but the convention is to use all lowercase, and hyphens for spaces.
